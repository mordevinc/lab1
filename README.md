# Трапеция
Репозиторий создан в рамках учебной работы по изучению платформы GitHub и практики использования Git для хранения, обновления и распространения исходного кода проекта.

# Цели и задачи работы
- Освоить базовые принципы работы с Git (коммиты, ветки, слияния, теги).

- Научиться использовать GitHub для хранения, обновления и распространения исходного кода.

- Получить практические навыки оформления репозитория (README.md, лицензия, структура).

- Реализовать алгоритмы для заданной геометрической фигуры в виде набора функций.

# Задание
Разработать алгоритм решения задачи по индивидуальному заданию:

Фигура: Трапеция

# Функции программы:

Вычисление периметра трапеции (сумма всех сторон)

Вычисление площади трапеции по формуле: S = ((a + b) / 2) × h

Вычисление длины средней линии трапеции: m = (a + b) / 2

Защита от ввода некорректных данных (отрицательные или нулевые значения сторон и высоты)
